## Development

Snapscope is a Progressive Web Application built with

- Typescript, React, NextJS running on Vercel
- Targets mobile devices, desktops and tablets
- Offline first design. Uses local storage.

```sh
# install nodejs and pnpm
asdf install

# install dependencies
pnpm install

# open claude
pnpm run c
```
